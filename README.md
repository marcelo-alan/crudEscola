# 🏫 CrudEscola

Sistema de gerenciamento escolar desenvolvido em **C#** com **Windows Forms**. O projeto permite o cadastro e gerenciamento de alunos, professores e turmas, facilitando o controle administrativo de uma instituição de ensino.

## 🚀 Funcionalidades
- Tela de Login segura para acesso ao sistema.
- Cadastro completo de Alunos e Professores.
- Gerenciamento de Turmas.
- Busca e filtragem de registros.
- Dashboard principal com navegação intuitiva.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C#
- **Interface:** Windows Forms (WinForms)
- **IDE:** Visual Studio 2022
- **Gerenciamento de Pacotes:** NuGet

## 📁 Estrutura do Projeto
O projeto segue uma organização profissional de pastas:
- `Classes/`: Modelos de dados e lógica de negócio.
- `Views/`: Telas da interface do usuário.
- `Resources/`: Imagens e ícones do sistema.

## 📝 Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/marcelo-alan/crudEscola.git`
2. Abra o arquivo `CrudEscola.sln` no Visual Studio.
3. Restaure os pacotes NuGet (se necessário).
4. Pressione `F5` para compilar e rodar.
